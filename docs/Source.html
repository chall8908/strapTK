listAttributes(addAttr) → {String}

Compiles all the HTML attributes and returns them in a manner acceptable for DOM insertion This method always tries to attach the ID and classes of the Panel If supplied string keys as arguments, it will attempt to add these keys to the attribute output in the following form: key='value_of_key' If the key has no value or is not defined, it is not added to the list. A key is considered to have no value if it is === "". If you need an attribute to have this value, you should manually add it to the list of attributes.
